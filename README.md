# Microsserviço (Cluster/Docker)

Um projeto baseado em um bootcamp da DIO.
Denilson Bonatti, Instrutor - Digital Innovation One

## Aprendizados

Aprendi bastante sobre como a criação de containers funciona de forma prática em microsserviços com cluster, entender como funciona o gerenciamento desses containers em diferentes servidores ao mesmo tempo, utilizando o proxy do nginx para o balanceamento de carga das requisição para todas as máquinas e o cluster (docker swarm) fazendo o gerenciamento dessas máquinas. 

## Images usadas

- mysql:5.7
- webdevops/php-apache:alpine-php7
- nginx
- my-proxy-app
