# Preparando o Container PHP

Isso é apenas um container exemplo, pode ser usado qualquer outro container de outra linguagem web.

## Criando o Volume do App

```bash
docker volume create app
```

### Passando o index.php para o volume

Passe o `index.php` para dentro do `app/_data`.
